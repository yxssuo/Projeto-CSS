# Curso Front-End

Este projeto foi desenvolvido como parte dos meus estudos em desenvolvimento web durante um curso de HTML5 e CSS3.

## Tecnologias utilizadas

- HTML5
- CSS3
- Estrutura com tags semânticas

## Objetivo

Praticar a criação de páginas web utilizando HTML semântico e aplicar estilização com CSS para construção de layouts.

## O que foi praticado

- Estruturação semântica do HTML
- Criação de seção hero com imagem de fundo
- Menu de navegação
- Organização de layout com CSS
